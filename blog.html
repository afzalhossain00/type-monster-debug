Difference between Local storage vs Session storage?

    Local Storage:
  • The data is stored in local storage for a lifetime unless the user manually deletes it from the browser.
  • Data can only be read on the client-side, not on the server-side
  • It's data in the browser can only be accessed via JavaScript and HTML5.
  • The storage capacity of local storage is 5MB/10MB.

  • The storage capacity of session storage is 5MB
